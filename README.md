# Logistics Management App

A comprehensive logistics management system built with Next.js, React, and Tailwind CSS.

## Features

- **User Authentication & Role-based Access**
  - Admin, Manager, Dispatcher roles
  - Protected routes based on permissions

- **Order Management**
  - Create, view, and manage orders
  - Order status tracking
  - Rider assignment

- **Rider Tracking**
  - Live rider location tracking
  - Rider status management
  - Performance analytics

- **Payment System**
  - Transaction management
  - Rider wallet system
  - Payout processing

- **Analytics & Reporting**
  - Comprehensive dashboards
  - Performance metrics
  - Area-wise analytics

- **Notifications**
  - Real-time notifications
  - Push notification support
  - Notification preferences

## Tech Stack

- **Frontend**: Next.js 14, React 18, Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Context API
- **Authentication**: JWT-based with localStorage
- **Notifications**: Web Notifications API
